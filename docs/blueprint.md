# **App Name**: VezaLeads

## Core Features:

- CRM Leads Display: Display a list of CRM leads in a grid layout, as in the given React code.
- Search Companies: Allow users to search for companies by name or other criteria.
- User settings: Provide a settings page for user to manage configurations and account details.

## Style Guidelines:

- Primary color: Orange (#FF5722), drawn directly from the component styling. Provides energy and immediacy to the application.
- Background color: Light gray (#F3F4F6), creates a clean, modern backdrop that contrasts with the primary color.
- Accent color: A muted blue-gray (#6B7280), for secondary actions and less critical UI elements. A cooler tone which offers a visual balance against the warmth of the orange.
- Font: 'Inter', a grotesque-style sans-serif font that looks modern and neutral; use for headlines and body text.
- Use Lucide React icons for navigation and actions. These icons are simple, modern, and consistent.
- Use a side navigation layout as defined in the user's original request to provide the best usability.
- Add subtle transitions and animations when switching between routes or when data is loading.
